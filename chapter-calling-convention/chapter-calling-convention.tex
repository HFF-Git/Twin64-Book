An exported procedure will have a different entry and ext sequence compared to a local procedure. The .ENTER and .LEAVE pseudo instructions will generate the respective code sequences.

The .CALLINFO pseudo instruction will describe the procedure.

The code snippet below is the maximum code to be generated.

\lstset{style=codesnippetstyle}
\begin{lstlisting}
			.CALLINFO ...		; procedure description
		
			;ENTER sequence
			ST SP,(SP) 			; save PSP
			ST RL, -8(SP)		; save RL
			ST DP, -16(SP)      ; save DP
			MFIA.S R1           ; get module LTAB address
			LD DP, <ofs>(R1)    ; load module DP
			LDO SP,<size>(SP) 	; allocate stack frame
			.
			.
			.
			.
			;LEAVE sequence
			LDO SP, -<size>(SP) ; deallocate stack frame
			LD  DP, -16(SP)     ; load previous DP
			LD  RL, -8(SP) 		; load return link
			BE (RL)				; external branch return
\end{lstlisting}

the DP value is taken from the first entry in the LTAB section for the module.
