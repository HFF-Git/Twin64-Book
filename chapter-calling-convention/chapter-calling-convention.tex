\section*{External Call - Method 1}
\addcontentsline{toc}{section}{External Call - Method 1}


A stub gets us from our own module to the target module. This is an unoptimized  sketch for now...


\lstset{style=codesnippetstyle}
\begin{lstlisting}
			LDIL 	R1, L%PLT           ; get linkage table					
			LDO     R1, R%PLT + ofs(R1) ; ofs is our entry
			ST      DP, -24(SP)		    ; save DP
			ST      RL, -16(SP)         ; save RL
			LD      DP, 16(R1)          ; load target DP
			LD      R1, 24(R1)          ; load target proc
			BE      (R1)                ; branch
            NOP							; filler
\end{lstlisting}


Target procedure return must restore DP and RL.

\lstset{style=codesnippetstyle}
\begin{lstlisting}

			; on leave 
			LDO 	SP, -frameSize(SP)	; cut back stack
			LD  	DP, -24(SP)        	; restore DP
			LD      RL, -16(SP)         ; restore RL
			BE		(RL)                ; return
\end{lstlisting}


\section*{External Call - Method 2}
\addcontentsline{toc}{section}{External Call - Method 2}

\lstset{style=codesnippetstyle}
\begin{lstlisting}

procName:   
			.CALLINFO					: options for proc
			.ENTER						: routine preamble
			
			; maximal code sequence generated by .ENTER
			LDIL 	R1, L%PLT           ; get our linkage table					
			LDO     R1, R%PLT + ofs(R1) ; ofs is our entry
			ST      DP, -24(SP)		    ; save DP
			ST      RL, -16(SP)         ; save RL
			LD      DP, 16(R1)          ; load target DP
			LDO 	SP, -frameSize(SP)	; allocate stack
			
			
			.LEAVE
			; maximal code sequence generated by .LEAVE
			LDO 	SP, -frameSize(SP)	; cut back stack
			LD  	DP, -24(SP)        	; restore DP
			LD      RL, -16(SP)         ; restore RL
			BE		(RL)                ; return
\end{lstlisting}
