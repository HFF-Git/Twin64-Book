\section*{EXIT — Leave Simulator}
\addcontentsline{toc}{section}{EXIT — Leave Simulator}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Syntax}
\texttt{exit | e [ <val> ]} \\
\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Description}

The \texttt{EXIT} command terminates the simulator.
An optional return value may be provided and is passed to the calling environment.

Both \texttt{EXIT} and its short form \texttt{E} are accepted as command aliases.

\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescOpEntry}{Example}

\begin{lstlisting}[style=iPageOpStyle]
(52) -> exit 100
\end{lstlisting}

\end{iPageDescOpEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Notes}
None.
\end{iPageDescEntry}
