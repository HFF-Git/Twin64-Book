\section*{HIST — List Command History}
\addcontentsline{toc}{section}{HIST — List Command History}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Syntax}
\texttt{hist [ <depth> ]} \\
\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Description}

The simulator command interpreter maintains a command history stack.
Each command, except for \texttt{DO} and \texttt{REDO}, is added to this stack.

The \texttt{HIST} command lists the contents of the history stack, starting
with the oldest entry and ending with the most recent one.

If an optional depth is specified, only the most recent \texttt{<depth>}
entries are displayed.

\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Example}

The following example shows the command history stack.
The \texttt{HIST} command itself is not added to the stack.

\begin{lstlisting}[style=iPageOpStyle]
(11) -> hist
[0]: env A_VAR 100
[1]: env
[2]: env A_VAR
[3]: help ptlb
[4]: wn cpu, 3
[5]: wn tlb, 3
[6]: wf
(7) ->
\end{lstlisting}

\end{iPageDescEntry}
