\section*{HALT — Halt a T64 module}
\addcontentsline{toc}{section}{HALT — Halt a T64 module}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Syntax}
\texttt{HALT [ <modNum> | ALL  ]} \\
\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Description}

The \texttt{HALT} command will stop a T64 module or the entire simulation. Modules running in a thread, such as processors and I7O modules, will enter the HALT state and wait for restarting.

\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescOpEntry}{Example}
\begin{lstlisting}[style=iPageOpStyle]

(5)->halt 3        # halt module 3
(6)->

(6)->halt all      # halt the entire simulation
(7)->

\end{lstlisting}
\end{iPageDescOpEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Notes}

None.

\end{iPageDescEntry}
