\section*{DMOD - List Module Info}
\addcontentsline{toc}{section}{DMOD - List Module Info}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Syntax}
\texttt{dmod [ <modNum> ]}
\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Description}

The \texttt{DM} command displays the configured modules of the Town64 system.
Each module is listed with its module number, type, hard physical address
(\texttt{HPA}), and the optional soft physical address range
(\texttt{SPA}) and size.

If no module number is specified, all modules are displayed. If a module
number is provided, only the selected module is shown.

\end{iPageDescEntry}
