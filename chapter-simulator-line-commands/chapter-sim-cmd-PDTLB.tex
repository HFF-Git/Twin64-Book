\section*{PDTLB — Purge data TLB}
\addcontentsline{toc}{section}{PDTLB — Purge data TLB}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Syntax}
\texttt{pdtlb [ <vAdr> ]}
\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Description}

The \texttt{PDTLB} command invalidates entries in the data Translation
Lookaside Buffer (TLB). If a virtual address \texttt{vAdr} is specified, only the
data TLB entry matching that virtual address range is purged. If no
address is supplied, all data TLB entries are invalidated.

\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescOpEntry}{Example}
\begin{lstlisting}[style=iPageOpStyle]

(3)-># purge the entire instruction TLB
(3)->pdtlb

(4)-># purge the instruction TLB entry covering virtual address 0x1000_0000
(4)->pdtlb 0x80_1000_0000

\end{lstlisting}
\end{iPageDescOpEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Notes}

Purging the data TLB does not affect the instruction TLB unless the processor implements a unified TLB. 

\end{iPageDescEntry}
