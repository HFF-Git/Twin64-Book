\section*{PITLB — Purge Instruction TLB}
\addcontentsline{toc}{section}{PITLB — Purge Instruction TLB}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Syntax}
\texttt{pitlb [ <vAdr> ]}
\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Description}

The \texttt{PITLB} command invalidates entries in the instruction Translation
Lookaside Buffer (TLB). If a virtual address \texttt{vAdr} is specified, only the
instruction TLB entry matching that virtual address range is purged. If no
address is supplied, all instruction TLB entries are invalidated.

\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescOpEntry}{Example}
\begin{lstlisting}[style=iPageOpStyle]

(3)-># purge the entire instruction TLB
(3)->pitlb

(4)-># purge the instruction TLB entry covering virtual address 0x1000_0000
(4)->pitlb 0x40_1000_0000

\end{lstlisting}
\end{iPageDescOpEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Notes}

Purging the data TLB does not affect the data TLB unless the processor implements a unified TLB.

\end{iPageDescEntry}
