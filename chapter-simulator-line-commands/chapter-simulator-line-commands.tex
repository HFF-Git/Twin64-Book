\section*{Command Syntax}
\addcontentsline{toc}{section}{Command Syntax}
	
Command syntax is described using the following conventions. Reserved words are written in uppercase letters. Parameters begin with a lowercase letters

Optional elements are enclosed in square brackets \texttt{[ ]}. Elements that may be repeated are enclosed in curly braces \texttt{\{ \}}.

Numeric values are shown with a \texttt{0x} prefix when expressed in hexadecimal notation. For improved readability, underscore characters may be used as digit separators within hexadecimal values.

String literals are enclosed in apostrophes (\texttt{' '}).
