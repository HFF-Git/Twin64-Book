\section*{REDO - redo command from history stack}
\addcontentsline{toc}{section}{REDO - redo command from history stack}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Syntax}
\texttt{REDO } \\
\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Description}

The \texttt{REDO} command displays a command from the history stack for editing. The edits are removing or inserting characters at the cursor position. A carriage return will then submit the command for execution.

\end{iPageDescEntry}
