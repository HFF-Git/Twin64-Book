\section*{WC — Set current window}
\addcontentsline{toc}{section}{WC — Set current window}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Syntax}
\texttt{wc <winNum>} \\
\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Description}

The \texttt{WC} command sets the current window to the specified window number.

Many window-related commands accept an optional window number parameter.
When this parameter is omitted, those commands operate on the current window.
The \texttt{WC} command therefore provides a convenient way to change the target
window for subsequent commands without explicitly specifying a window number
each time.

If the specified window number does not exist, the command fails and the current
window remains unchanged.

\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescOpEntry}{Example}
\begin{lstlisting}[style=iPageOpStyle]
wn mem, 0x1000          # Create a window (window number 1).
wn mem, 0x2000          # Create another window (window number 2).
                        # Window 2 is the current window.
wc 1                    # Set the current window to window 1.
\end{lstlisting}
\end{iPageDescOpEntry}
