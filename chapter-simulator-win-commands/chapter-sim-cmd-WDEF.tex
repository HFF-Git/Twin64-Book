\section*{WDEF — Set window defaults}
\addcontentsline{toc}{section}{WDEF — Set window defaults}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Syntax}
\texttt{wdef [ <start> [ , <end> ]] | ALL} \\
\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Description}

Windows are created with default attributes, such as the number of lines to display or the numeric display radix. The \texttt{WDEF} command resets these attributes to their default values for a single window, a range of windows, or all windows. 

If no parameters are specified or the \texttt{ALL} is used, all windows are reset to their default attributes.

\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescOpEntry}{Example}
\begin{lstlisting}[style=iPageOpStyle]
wdef                # Reset all windows to default attributes.
wdef 3, 7           # Reset windows 3 through 7 defaults.
\end{lstlisting}
\end{iPageDescOpEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Notes}
None.
