\section*{WSD — Disable Window Stack}
\addcontentsline{toc}{section}{WSD — Disable Window Stack}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Syntax}
\texttt{wsd <stackNum> | ALL } \\
\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Description}

The \texttt{WSD} command disables one or more window stacks, removing all
windows in the stack from the visible screen. Windows within a disabled stack are not deleted. All attributes, including layout, size, and position, are preserved.  

The keyword ALL disabled all window stacks. If the stack hosting the current window is disabled, the previous window is set as current window.

\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Notes}
None.
\end{iPageDescEntry}
