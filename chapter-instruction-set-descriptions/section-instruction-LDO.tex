\pagebreak
\section*{LDO — Load Offset}
\addcontentsline{toc}{section}{LDO — Load Offset}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Syntax}
\texttt{LDO [.B/H/W/D] RegR, ofs(RegB)}
\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Format}
The \texttt{LDO} instruction uses the following instruction formats: \\

\begin{flushleft}
\drawInstrFormat
    {0,1,3,5,6.5,8.5,9.5,16}
    {\OpCodeGrpALU,\OpCodeLDO,RegR,0,RegB, dw, ofs}
\end{flushleft}

\begin{flushleft}
    \drawInstrFormat
        {0,1,3,5,6.5,8.5,9.5,11.5,16}
        {\OpCodeGrpALU,\OpCodeLDO, RegR, 1, RegB, dw, RegX, 0}
    \end{flushleft}
    
\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Description}
The \texttt{LDO} instruction loads an offset into a general register. Two formats are available: the immediate-offset format and the register-indexed format. In the immediate-offset format, the signed immediate offset is shifted left by the \texttt{dw} field of the instruction and added to \texttt{RegB} to form the effective address. In the register-indexed format, \texttt{RegX} is added to \texttt{RegB} to form the effective address. The computed value is then stored in \texttt{RegR}.

\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescOpEntry}{Operation}
\begin{lstlisting}[style=iPageOpStyle]
RegR <- addOfs32( RegB, signExtend( ofs, 13 ) << dw );
RegR <- addOfs32( RegB, ( RegX << dw ));
\end{lstlisting}
\end{iPageDescOpEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Exceptions}
None.
\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Notes}

The \texttt{LDO} instruction is often used to load immediate values. 

\begin{verbatim}
    LDI Rm, val -> LDO Rm, val( R0 )
\end{verbatim}

If value to load is larger than the 13-bit offset, a sequence of instruction is however necessary.


\end{iPageDescEntry}
