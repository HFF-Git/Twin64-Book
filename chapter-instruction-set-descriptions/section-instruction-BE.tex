\section*{BE - Branch External}
\addcontentsline{toc}{section}{BE - Branch External}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Syntax}
\texttt{BE RegB [, ofs ]}
\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Format}
The \texttt{BE} instruction uses the signed immediate S15 format. \\

\begin{flushleft}
\drawInstrFormat
    {0,1,3,5,6.5,8.5,16}
    {\OpCodeGrpBR,\OpCodeBV, RegR, 0, RegB, ofs}
\end{flushleft}
\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescEntry}{Description}
The \texttt{BE} instruction performs an unconditional branch to another segment. 


Before branching, the next instruction address (\texttt{IA + 4}) is stored in \texttt{RegR}.  

Because \texttt{BE} is a segment base relative branch, branching to a page with a different privilege level is possible.  
Branching from a lower privilege level to a higher one triggers an instruction protection trap.  
Branching from a higher privilege to a lower privilege level updates the privilege level in the processor status register.  
If no change is required, the privilege level remains unchanged.  


\end{iPageDescEntry}

%--------------------------------------------------------------------------------
\begin{iPageDescOpEntry}{Operation}
\begin{lstlisting}[style=iPageOpStyle]
RegR <- addOfs32( PSW.[IA], 4 );
PSW.[IA] <- computeTarget( RegB, RegX );
\end{lstlisting}
\end{iPageDescOpEntry}
